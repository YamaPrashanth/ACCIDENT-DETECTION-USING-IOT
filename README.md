# ACCIDENT-DETECTION-USING-IOT

# Project Overview

This project proposes an IoT-based Accident Detection System using Arduino Uno, GSM, GPS, and vibration sensors. The system detects vehicle accidents and automatically sends alert messages along with live location coordinates to emergency contacts, helping provide faster medical assistance and improving road safety.

# Components Used
- Arduino Uno
- GSM Module
- GPS Module
- Vibration Sensor
- LCD Display
- Buzzer
- Power Supply Circuit
  
# Working Principle
- The vibration sensor detects accident impact.
- Arduino processes the sensor data.
- If accident intensity exceeds the threshold:
  - Buzzer turns ON
  - GPS captures live location
  - GSM sends emergency alert message
- LCD displays accident status.
